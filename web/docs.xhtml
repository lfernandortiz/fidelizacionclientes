Plataforma Puntos Farmanorte

Puntos Farmanorte es la plataforma administrativa del programa de fidelización de cliente de Dromedicas del Oriente. Esta documentacion se divide en 3 partes. La primera explica los requerimientos de implementacio y puesta en marcha de la aplicacion. La Segunda parte comprende la documentacion de usuario. La tercera parte es la informacion para desarrolladores de la API de Puntos Farmanorte

Esta guia propone las opciones de configuracion y puesta en marcha de la aplicación.

Theme

Omega layout depends on the Omega theme of PrimeFaces that is available since 6.0. Setup is simple as defining primefaces.THEME context parameter in web.xml such as "omega".

Menu

Menu is a regular JSF component that supports PrimeFaces MenuModel API allowing both declarative and programmatic approaches. State is saved via a cookie and to remove it define a widgetVar to the menu and call PF('yourwidgetvar').clearMenuState(). To use the menu add the omega-menu.jar to your classpath and import the http://primefaces.org/omega namespace.

Menu becomes collapsed automatically in screens smaller than 1024px and it can be toggled on demand using the menu button at the top bar in larger screens as well. In a large screen, if you'd like to display the menu as collapsed by default, sidebar-inactive-l class to the main wrapper div (<div class="wrapper">).

xmlns:po="http://primefaces.org/omega">

<po:menu>
</po:menu>
